# ✨ Localization Workflow

Every project follows the same workflow:

```text
Original
      ↓
Script Translate
      ↓
Manual Review
      ↓
Terminology Check
      ↓
UI & Context Verification
      ↓
Final Localization
```

Machine translation is only used as a tool.

Every localization is manually reviewed and improved before delivery.
